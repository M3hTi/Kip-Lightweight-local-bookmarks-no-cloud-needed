## Getting Started

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Start the JSON server (for bookmark storage):

```bash
npm run server
```

4. Start the development server:

```bash
npm run dev
```

## Default Login Credentials

- Username: admin
